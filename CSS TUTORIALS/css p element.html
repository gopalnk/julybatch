css pseudo elements is used to style specified parts of an elements

    syntsx==selector::pseudo elements{} 
  types of pseudo elements:

1]first line--style to first line    p::first-line{}
2]first letter--style to first letter     p::first-letter{}
3]before/after--insert before/after content in given element content     h1::before/after{content:'property'}
4]marker--style to markers in list elements    ::marker{ }
5]selection--to style the our selected portion in tha data  ::selection{ }
