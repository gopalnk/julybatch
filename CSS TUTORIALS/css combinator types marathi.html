css combinator 

        
combinator have 4 types

        1.descendent[वंशज]
        2.child[मुले]
        3.adjuscent sibling[लागतचा भाऊ/बहीण]
        4.general siblings [सर्व भाऊ/बहीण]

        
1.descendent<वंशज> combinator[div space p ]

         एखाद्या tag च्या आतील जेवढे पण tag असतील ते सर्व tag <वंशज>
            e.g. div या मधील सर्व tag 


            
2.child [div>p]


body मधील जेवढे div tag असतील त्यामधील p tag ला style apply होते 


         
3.adjuscent siblings [div+p]

         div tag नंतरचा लगेच येणारा p tag म्हणजेच div चा adj sibling means लागतचा भाऊ/बहीण 


         
4.general siblings [div~p]

         div tag नंतरचे येणारे सर्व p tag म्हणजेच div चे general siblings होय.
